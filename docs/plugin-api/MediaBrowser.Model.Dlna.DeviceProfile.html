    Class DeviceProfile

    A DeviceProfile represents a set of metadata which determines which content a certain device is able to play.
    Specifically, it defines the supported ContainerProfiles and CodecProfiles (video and/or audio, including codec profiles and levels) the device is able to direct play (without transcoding or remuxing), as well as which TranscodingProfiles in case it isn't.
