Chat

We primarily use Element to access the Matrix network.

  • #jellyfin:matrix.org: General chat related to the project
  • #jellyfin-announce:matrix.org: Announcements for releases
  • #jellyfin-troubleshooting:matrix.org: User troubleshooting
  • #jellyfin-dev:matrix.org: Main room for development communication
  • #jellyfin-dev-client:matrix.org: Client and Web development
  • #jellyfin-dev-android:matrix.org: Android, Android TV and Fire TV development
  • #jellyfin-offtopic:matrix.org: Chat about anything
  • #jellyfin-translate:matrix.org: Help translate Jellyfin

IRC

All channels are bridged from Matrix to Freenode for convenience.

  • #jellyfin - webchat
  • #jellyfin-announce - webchat
  • #jellyfin-troubleshooting - webchat
  • #jellyfin-dev - webchat
  • #jellyfin-dev-client - webchat
  • #jellyfin-dev-android - webchat
  • #jellyfin-offtopic - webchat

Discord (Beta)

For convenience, we have a Discord server that is bridged to the chats above. Note that Matrix is still the preferred chat platform, and Discord messages may be missed due to bridge instability.
