Chat

To chat, we primarily use Riot to access the Matrix network.

  • #jellyfin:matrix.org: General chat related to the project
  • #jellyfin-announce:matrix.org: Announcements for releases
  • #jellyfin-troubleshooting:matrix.org: User troubleshooting
  • #jellyfin-dev:matrix.org: Main room for development communication
  • #jellyfin-dev-client:matrix.org: Client and Web development
  • #jellyfin-dev-android:matrix.org: Android, Android TV and Fire TV development
  • #jellyfin-offtopic:matrix.org: Chat about anything
